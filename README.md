## Uptime Checker

Система мониторинга доступности веб-ресурсов, состоящая из двух микросервисов.

### Описание сервисов

Watcher: Принимает таски для опроса сайтов по кафке. 
Использует горутины для параллельных HTTP-запросов и отправляет результаты в API.
Реализована фича с переменным количеством горутинов в зависимости от нагрузки процессора

API: Центральный узел, который принимает отчеты от воркеров, обрабатывает и сохраняет историю проверок.

### Технологический стек

- Язык: Golang
- Сборка: Docker (Multi-stage builds, Alpine)
- Оркестрация: Docker Compose, Nginx
- Взаимодействие: Kafka
- БД: PostgreSQL + GORM + Goose
- Gin

### Интересные фичи
- Динамическое управление пулом горутин на основе runtime.NumCPU() и текущей нагрузки (CPU Load Average)
- Nginx Load Balancer
- Старался сделать возможность горизонтального масштабирования
