## Uptime Checker

Система мониторинга доступности веб-ресурсов, состоящая из двух микросервисов.

### Описание сервисов

Watcher: Принимает таски для опроса сайтов по кафке. Использует горутины для параллельных HTTP-запросов и отправляет результаты в API.

API: Центральный узел, который принимает отчеты от воркеров, обрабатывает и сохраняет историю проверок.

### Технологический стек

- Язык: Go (Golang)
- Сборка: Docker (Multi-stage builds, Alpine)
- Оркестрация: Docker Compose
- Взаимодействие: Kafka
- БД: PostgreSQL + GORM
